# Markdown编写规范

数字、代码、特殊符号 都要前后加空格

标题不使用 `###` , 通常只使用 `##`、`####`, 最好不要超过两级, `#` 用于文章名称

逗号使用英文逗号 `,` 逗号后面要加一个空格

标签、插件名称要强调的话需要用 ` `两个转义符括起来, 如 `<link>`

如果是是中文, 就用 英文双引号 括起来, 如 “模板”

图片使用 Markdown 格式读取图片, 写文章的时候把文件都写到 public 文件夹中

一些注解、【注意】、【提示】都放在 `>` 引用符号中

图片居左, 在 `![名称](url) `后面 按 "tab键"
